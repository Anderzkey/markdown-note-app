## Features to Test

- Click the **Edit** button to enter edit mode
- Type some changes in the textarea
- Word count and character count update live
- Click **Preview** to toggle between editor and preview
- Click **Save** to save changes
- Click **Cancel** to discard and exit
- Press `Ctrl+E` to toggle edit mode (or `Cmd+E` on Mac)
- Press `Ctrl+S` to save while editing
- Press `Escape` to exit edit mode
